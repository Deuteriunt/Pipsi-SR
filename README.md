<h1 align="center">Features</h1>

#### Player

- Noclip (toggle : `V`)

#### Battle

- Battle Speed

#### World

- Chest Teleport (hold : `F4`)
- Quest Teleport (hold : `F2`)
- Talk Speed
- Dump Enemies

#### Visuals

- Fov changer
- Peeking
